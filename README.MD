## About

**Adtide** is an advanced ad server management platform that helps you manage publishers, partners, and ad units all in one place. Streamline your operations and maximize your revenue with our powerful dashboard and comprehensive management tools.

## Features

- Landing Page
- Authentication
- Dashboard
- Partner Management
- Publisher Management
- Ad Unit Management
- Partner List
- Publisher List
- Ad Unit List

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/woabu0/adtide.git
   ```

2. Navigate to the project directory:

   ```bash
   cd adtide
   ```

3. No dependencies required - this is a static HTML/CSS/JavaScript project.
